# Fashion Marketplace UX Builder

This is a Figma development plugin that builds a 1440 x 1024 fashion marketplace UI/UX concept.

It creates:

- A desktop fashion home screen with a real marketplace structure: trust bar, logo, search, location, account, wishlist, cart, category navigation, editorial hero, trend chips, curated rails, and product discovery.
- A listing screen with Myntra-style fashion filters and Amazon-style ratings, delivery promises, returns, and verified brand cues.
- Product, Cart, and Checkout frames with size selection, fit guide, offers, pincode delivery check, reviews, secure checkout, and order summary.
- Click-through prototype links for Home -> Listing -> Product -> Cart -> Checkout.

## Run In Figma

1. Open Figma Desktop.
2. Go to `Plugins > Development > Import plugin from manifest...`.
3. Select `manifest.json` from this folder.
4. Run `Fashion Marketplace UX Builder` from `Plugins > Development`.

The plugin creates a new page named `Fashion Marketplace UX` in the active Figma file.
